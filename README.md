# YOLOv8 Object Detection Model

## Overview
This project implements a real-time object detection system using YOLOv8, Python, and OpenCV. The system detects humans and various objects through webcam input and displays bounding boxes with confidence scores.

## Features
- Real-time object detection
- Human detection
- Confidence score display
- Webcam integration
- Multiple object recognition

## Technologies Used
- Python
- YOLOv8
- OpenCV
- NumPy
- Computer Vision

## Project Outcome
Successfully developed a real-time object detection model capable of identifying humans and various objects with high accuracy.

## Future Scope
- Crowd density estimation
- Enhanced object tracking
- Smart surveillance applications
